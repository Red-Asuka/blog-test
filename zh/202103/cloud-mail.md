尊敬的用户，您好！

感谢您对 EMQ X Cloud 产品的使用和关注！ 

EMQ X Cloud 作为一款全托管的 MQTT 5.0 云服务，自诞生以来受到大量用户的关注、鼓励与认可，同时也收到了很多用户的反馈。帮助用户获得成功是我们产品的首要目标，您的每一个建议和意见都是我们持续进步的动力。经过团队不断地努力，我们很高兴的通知您：EMQ X Cloud 3.0.4 版本正式上线！
